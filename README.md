# Levler Chat UI

A modern chat UI application built with React, TypeScript, and Vite.

## Quick Start

```bash
git clone https://github.com/eylulbalcilar/levler-chat-ui.git
cd levler-chat-ui
npm install
npm run dev
```

Then open http://localhost:5173/

## Available Scripts

- `npm run dev` - Start dev server
- `npm run storybook` - Start Storybook (http://localhost:6006/)
- `npm run build` - Build for production
- `npm run test` - Run tests with Vitest

## Tech Stack

- React 18 + TypeScript
- Vite
- Storybook 10
- Vitest + Testing Library
